# compass

Webアプリで自宅の方向を指し示すコンパスです。ブラウザの位置情報とデバイスの向きを利用し、保存した自宅の緯度経度へ矢印が向きます。

## 使い方
1. ブラウザに位置情報とデバイスの向きの利用を許可します。
